# Superstore Sales Analytics Dashboard

## Project Overview

This project analyzes retail sales data using Python for data preprocessing and Power BI for interactive business intelligence visualization. The dashboard provides insights into sales performance, regional trends, product profitability, and customer behavior.

## Tools & Technologies

* Python (Pandas, NumPy, Matplotlib)
* Power BI
* Excel
* Jupyter Notebook

## Data Preparation (Python)

* Data cleaning and preprocessing
* Handling missing values
* Feature formatting for analysis
* Exported cleaned dataset for Power BI

File: `analysis.ipynb`

## Power BI Dashboard Pages

### 1. Sales Overview

* Total Sales, Profit, Orders, Profit Margin
* Monthly Sales Trend
* Sales by Category
* Interactive filters (Region, Category, Year)

### 2. Regional Analysis

* Sales and Profit by Region
* Region profitability comparison
* Geographic visualization

### 3. Product Performance

* Top 10 Products by Sales
* Sales vs Profit analysis
* Loss-making product identification

### 4. Customer Analysis

* Top Customers
* Segment-wise Sales distribution
* Customer profitability insights


## Business Insights Generated

* Identified high-sales but low-profit products
* Detected loss-making items
* Analyzed regional performance differences
* Highlighted key revenue-generating customers
